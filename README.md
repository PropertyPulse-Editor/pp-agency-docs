# PP Agency Documentation Website

PP Agency is the official employee application for Property Pulse, owned and operated by Exodus Media. This repository folder contains the lightweight public documentation website used for app support, privacy disclosures, terms, and GitHub Pages hosting.

The website is built with static HTML and CSS only. No JavaScript, frameworks, build tools, or generated assets are required.

## Folder Structure

```text
PP-Agency-Docs/
  index.html      Home page and application overview
  privacy.html    Privacy Policy for Google Play and app users
  terms.html      Terms & Conditions for authorized employee use
  support.html    Support and company contact information
  style.css       Shared responsive styling
  README.md       Project documentation
  assets/
    logo.png      Property Pulse / PP Agency logo and favicon source
```

## Pages

- Home: overview of PP Agency and its internal employee-only purpose.
- Privacy Policy: covers authentication, customer data, document uploads, camera, photos/media, location, biometric login, Supabase storage, security, user rights, and contact details.
- Terms & Conditions: covers authorized employee use, ownership, responsibilities, acceptable use, intellectual property, account suspension, liability, and governing law.
- Support: lists email, phone, official website, business hours, and company details.

## GitHub Pages Deployment

1. Push the `PP-Agency-Docs` folder contents to a GitHub repository.
2. Open the repository settings on GitHub.
3. Go to Pages.
4. Select the branch and folder that contain `index.html`.
5. Save the configuration and open the published GitHub Pages URL.

The site is GitHub Pages compatible because all links are relative and all pages use static files only.
